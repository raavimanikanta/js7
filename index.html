What is an array

Array is a data structure which consists of collection of elements.This elements can be accessed by individual index or keys

Example: 
        const array=[1,2,3,4]
        
        for(i=0 ;i< array.length ;i++) {
            console.log(array[i])
        }
        output:1 2 3 4

Map Method

Map is an array method used to itterate over the elements of the array and it returns the new array.

Example: 
        const array=[1,2,3,4,5]
        const NewArray=array.map((result)=>{
                 result+2
        })
        console.log(NewArray)

        Output:[3,4,5,6,7]

Filter Method

Filter is an array method used to itterate over the elements of the array with conditions and it returns the new array.The elements which are out of the conditions are not displayed in the new array

Example: 
        const array=[1,2,3,4,5]
        const NewArray=array.filter((result)=>
                 result>1
        )
        console.log(NewArray)

        Output:[2,3,4,5]

Reduce Method

Reduce is an array method used to itterate over the elements of the array and it returns the new array with the sum of elements .The call back function takes two parameters accumulative and current values.In this we have to set the initial value and to that initial value the sum of elements value will be added .

Example: 
       var ArrayReduce=[20,30,40,50]
       var ArrayReduce1=ArrayReduce.reduce((acc,curr)=>
          acc+curr 
       ,10);
       console.log(ArrayReduce1)

        Output:150

Push Method

Push is an array method used to add the element at the end of the array

Example:
        const array=[1,2,3,4,5]
        array.push(6);
        console.log(array);
        Output:[1,2,3,4,5,6] //New element will be added at the last.

Pop Method

Pop is an array method used to remove the element at the end of the array

Example:
        const array=[1,2,3,4,5]
        array.pop(5);
        console.log(array);
        Output:[1,2,3,4] //The element gets deleted  from  the last.

unshift Method

unshift is an array method used to add the element at the beginning of the array

Example:
        const array=[1,2,3,4]
        array.unshift (5);
        console.log(array);
        Output:[5,1,2,3,4] //The element gets added at  the first index

shift Method

shift is an array method used to rempve the element from the beginning of the array

Example:
        const array=[1,2,3,4]
        array.shift (1);
        console.log(array);
        Output:[2,3,4] //The element gets deleted  from  the first index

Some Method

Some is an array method checks the elements from the given condition and returns Boolen value.if condition satisfied for one element it will return true

Example:
        var arraySome=[1,2,3,4,5]
        var ArraySome1=arraySome.some(result=>result>1)
        console.log(ArraySome1)
    OutPut:true.

Every is an array method checks the elements from the given condition and returns Bool

Every Method

en value.if condition satisfied for all elements then it will return true or it will return false

Example:
        var arraySome=[1,2,3,4,5]
        var ArraySome1=arraySome.every(result=>result>1)
        console.log(ArraySome1)
    OutPut:false.

Slice Method

Slice is a array method which returns selected elements from the array as per the starting index and end index.

Example:
        var arr=[1,2,3,4];
        var data=arr.slice(0,3);
        console.log(data);

        Output:
        1 2 3
